# Money API Headless (Cho Obsidian)

Đây là API back-end (đã tái thiết) cho hệ thống Money Flow.
Repo này KHÔNG chứa UI. UI duy nhất là Obsidian.

## Endpoints Hiện có (Reference Data)
* `/api/accounts`
* `/api/people`
* `/api/categories`
* `/api/shops`

## Endpoints Mới (Transaction Engine - Cần xây dựng)
* `/api/transactions` (Xử lý giao dịch mới)
* `/api/linked-txn` (Xử lý workflow hủy đơn, hoàn tiền...)
